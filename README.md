# Porpeta's Sunny Quest

A tiny browser game for Porpeta, with a real Porpeta title video, gentle music, and one very important sunny nap.

Play it here: https://dhasakgbb.github.io/porpeta-sunny-quest/

Play through three warm little rooms:

- Sneak through the kitchen to reach the food bowl.
- Catch five red laser dots in the living room.
- Curl up in the bedroom sun patch for the final purr.

## Run Locally

```bash
python3 -m http.server 5173
```

Then open `http://localhost:5173`.

## Controls

- Move: WASD or arrow keys
- Sneak: Shift or B
- Pounce: Space
- Fullscreen: F
- Sound: toggle with the button in the lower-right corner
